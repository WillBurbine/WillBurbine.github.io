My Javascript programs!

This page is where all of my Javascript progams will go. These progams were
completed for my CSC 106 class from the University of Rhode Island.

Burbine_racingBitmojiFun

This is a race between a bitmoji of myself that
I created and a frog. Refresh the page to see a
differnet speed!

Burbine_DancingBitmojiFun

This is a drawing of a bitmoji I made. It is called
dancing bitmoji because every time you refresh the
page the bitmoji is in a different spot.

Burbine_decision

This is a decision tree program.
Click on the answers to get results.
Restart to get a different result.

Burbine_Follow

This program has the bitmoji I made follow
your cursor. If the bitmoji hits any point on
the top edge then the program ends.
