Welcome to my My Final Project!

For this project me and my partner made a snake game. The first screen you
see is the start screen where you can select play or palettes. If you click play
you go straight to the game, but if you click palettes you get the
option to select there colors which change the look of the game. To start playing
the game you use the arrow keys to move the snake around. To make the
snake bigger you collect the dots that apear. The snake can warp from one side of
the stage to the other so the way you lose the game is if the
snake hits itself. When you lose the game you will be taken to a game
over screen where you can see your score as well as the high score
and there is a reset button so you can play again.
